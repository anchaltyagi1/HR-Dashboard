# 📊 HR Analytics Dashboard (Excel)

An interactive HR analytics dashboard built entirely in **Microsoft Excel** to track and visualize workforce metrics.  
This project demonstrates how HR teams can leverage Excel features like **Pivot Tables, charts, slicers, and conditional formatting** to monitor workforce composition, identify trends, and support strategic decision-making.

---

## 🔑 Key Features
- 📈 **Headcount Growth (2020–2025):** Tracks onboarding, attrition, transfers, and overall growth.
- 👥 **Diversity Analysis:** Gender distribution across employees.
- 🏢 **Department & Job Levels:** Employee breakdown by department and role.
- ⏳ **Stability Insights:** Tenure analysis for retention monitoring.
- 📑 **Manager Reporting:** Headcount reporting by supervisors.
